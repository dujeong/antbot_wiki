## Launch 의존성 그래프

`bringup.launch.py`가 전체 시스템을 순차적으로 실행합니다:

<FileTree>

- bringup.launch.py **전체 시스템**
  - robot_state_publisher.launch.py URDF -> /tf, /tf_static
  - controller.launch.py ros2_control + swerve controller
  - imu.launch.py 6축 IMU (200 Hz)
  - lidar_2d.launch.py 2x COIN D4 (USB Serial)
  - lidar_3d.launch.py Vanjee WLR-722 (Ethernet)
  - ublox_gps_node-launch.py u-blox GNSS
  - camera.launch.py V4L2 + USB + RGB-D
  - teleop_joy.launch.py DualSense 조이스틱

</FileTree>

## 하드웨어 통신 구조

ANTBot의 모든 하드웨어 통신은 **Dynamixel Protocol 2.0**을 통해 이루어집니다:

| 연결 | 장치 | 포트 | 보드레이트 | ID |
| :--- | :--- | :--- | :--- | :---: |
| `RCU` | ANT-RCU 보드 | `/dev/ttyUSB0` | 4,000,000 bps | 200 |
| `IMU` | IMU 보드 | `/dev/ttyUSB1` | 4,000,000 bps | 200 |

### ros2_control 인터페이스

<Tabs>
  <TabItem label="Command Interfaces">
    | Joint | position | velocity | acceleration |
    | :--- | :---: | :---: | :---: |
    | `steering_{corner}_joint` | O | opt | opt |
    | `wheel_{corner}_joint` | - | O | O |
  </TabItem>
  <TabItem label="State Interfaces">
    | Joint | position | velocity | effort |
    | :--- | :---: | :---: | :---: |
    | `steering_{corner}_joint` | O | - | current |
    | `wheel_{corner}_joint` | encoder | O | current |
  </TabItem>
</Tabs>

:::tip
`corner`는 `front_left`, `front_right`, `rear_left`, `rear_right` 중 하나입니다.
:::
