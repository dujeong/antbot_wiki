## 서비스

<Tabs>
  <TabItem label="적재함 제어">
    **`cargo/command`** — `antbot_interfaces/CargoCommand`

    | 필드 | 타입 | 값 | 설명 |
    | :--- | :--- | :--- | :--- |
    | Request: `operation` | uint8 | 0 = LOCK | 적재함 잠금 |
    | | | 1 = UNLOCK | 적재함 해제 |
    | Response: `success` | bool | - | 성공 여부 |
    | Response: `message` | string | - | 상태 메시지 |

    ```
    # 적재함 잠금
    ros2 service call /cargo/command antbot_interfaces/srv/CargoCommand "{operation: 0}"

    # 적재함 해제
    ros2 service call /cargo/command antbot_interfaces/srv/CargoCommand "{operation: 1}"
    ```
  </TabItem>
  <TabItem label="전조등">
    **`headlight/operation`** — `std_srvs/SetBool`

    | 동작 | 명령 |
    | :--- | :--- |
    | 전조등 켜기 | `data: true` |
    | 전조등 끄기 | `data: false` |

    ```
    # 전조등 ON
    ros2 service call /headlight/operation std_srvs/srv/SetBool "{data: true}"

    # 전조등 OFF
    ros2 service call /headlight/operation std_srvs/srv/SetBool "{data: false}"
    ```
  </TabItem>
  <TabItem label="와이퍼">
    **`wiper/operation`** — `antbot_interfaces/WiperOperation`

    | 필드 | 타입 | 값 | 설명 |
    | :--- | :--- | :--- | :--- |
    | Request: `mode` | uint8 | 0 = OFF | 와이퍼 정지 |
    | | | 1 = ONCE | 1회 작동 |
    | | | 2 = REPEAT | 반복 (주기: cycle_time) |
    | Request: `cycle_time` | float32 | 기본 1.0 | 반복 주기 (초) |

    ```
    # 와이퍼 1회 작동
    ros2 service call /wiper/operation antbot_interfaces/srv/WiperOperation "{mode: 1}"

    # 와이퍼 2초 간격 반복
    ros2 service call /wiper/operation antbot_interfaces/srv/WiperOperation "{mode: 2, cycle_time: 2.0}"
    ```
  </TabItem>
  <TabItem label="컨트롤 테이블">
    **`DirectRead`** / **`DirectWrite`** 서비스를 통해 RCU 컨트롤 테이블 아이템에 직접 접근할 수 있습니다.

    ```
    # 배터리 잔량 읽기
    ros2 service call /direct_read antbot_interfaces/srv/DirectRead \
      "{item_name: 'Battery_Percentage'}"

    # LED 모드 설정
    ros2 service call /direct_write antbot_interfaces/srv/DirectWrite \
      "{item_name: 'Led_Flag_Mode', data: 2}"
    ```

    :::tip
    컨트롤 테이블 전체 아이템 목록은 [5.1 컨트롤 테이블](/antbot_wiki/development-guide/control-table/) 페이지를 참조하세요.
    :::
  </TabItem>
</Tabs>
