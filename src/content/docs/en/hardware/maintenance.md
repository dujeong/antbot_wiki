## 5.1 Firmware Diagnostics

When RCU board diagnostics are needed, use the provided **Dynamixel Wizard 2.0** following these steps:

1. Connect to the robot PC
2. Launch Wizard and scan ports (`/dev/ttyUSB0` connection)
3. Verify RCU board communication
