PS5 DualSense 컨트롤러를 사용해 ANTBot을 수동 조작할 수 있습니다.

![Controller Layout](../../../assets/images/joystick_dualsense(en).png)

## 연결 방법

1. DualSense 컨트롤러를 **USB 포트 1 (파란색)**에 유선 연결합니다
2. 자동으로 컨트롤러를 인식합니다 (약 2~3초 소요)
3. 컨트롤러 LED가 점등되면 사용 준비 완료입니다

## 조작 방법

### 이동 제어

| 입력 | 동작 | 비고 |
| :--- | :--- | :--- |
| **좌측 스틱 Y축** | 전진 / 후진 | 최대 1.0 m/s |
| **좌측 스틱 X축** | 커브 주행 | 전진/후진 중일 때만 동작 |
| **L2 트리거** | 좌측 제자리 회전 | 정지 상태에서만 동작 |
| **R2 트리거** | 우측 제자리 회전 | 정지 상태에서만 동작 |

### 주행 모드

ANTBot은 속도에 따라 두 가지 주행 모드로 자동 전환됩니다:

- **커브 주행** (`|vx| >= 0.05 m/s`): 좌측 스틱 X축이 각속도를 제어하며, 스티어링 한계(-60° ~ +60°) 내에서 동작합니다
- **제자리 회전** (`|vx| < 0.05 m/s`): L2/R2 트리거로 제자리 회전을 제어합니다

### 기능 버튼

| 버튼 | 기능 |
| :--- | :--- |
| **Triangle** | 속도 레벨 UP (1~9) |
| **Cross** | 속도 레벨 DOWN (1~9) |
| **Square** | 적재함 잠금 (Lock) |
| **Circle** | 적재함 해제 (Unlock) |
| **L1** | 전조등 토글 (ON/OFF) |
| **R1** | 와이퍼 토글 (ON/OFF) |

### 속도 레벨

속도 레벨은 1~9 단계로 조절할 수 있으며, 기본값은 **3**입니다.

| 레벨 | 최대 선속도 | 최대 각속도 |
| :---: | :---: | :---: |
| 1 | 0.11 m/s | 0.11 rad/s |
| 3 (기본) | 0.33 m/s | 0.33 rad/s |
| 5 | 0.56 m/s | 0.56 rad/s |
| 9 (최대) | 1.0 m/s | 1.0 rad/s |

## 키보드 원격 조종

컨트롤러가 없는 경우, SSH 접속 후 키보드로 조종할 수 있습니다:

```
ros2 run antbot_teleop teleop_keyboard
```

| 키 | 동작 |
| :---: | :--- |
| `W` / `X` | 전진 / 후진 |
| `A` / `D` | 좌측 / 우측 이동 (스트레이프) |
| `Q` / `E` | 반시계 / 시계 회전 |
| `S` | 정지 |
| `1` ~ `9` | 속도 레벨 |
| `ESC` | 종료 |

:::tip
키보드 텔레오퍼레이션은 전방향(holonomic) 이동을 지원합니다. `A`/`D` 키로 바퀴 방향 전환 없이 좌우로 이동할 수 있습니다.
:::
