ANTBot을 수령하셨다면, 아래 순서에 따라 개봉 및 구성품을 확인해 주세요.

## 구성품 확인

패키지에 포함된 구성품을 확인합니다:

| 구성품 | 수량 | 비고 |
| :--- | :---: | :--- |
| ANTBot 본체 | 1 | - |
| 수동 충전기 | 1 | HMI 패널 충전 단자 연결용 |
| USB 케이블 (Type-C) | 1 | 컨트롤러 연결용 |
| 스타트 가이드 | 1 | 간략 사용 설명서 |

:::note[옵션 구성품]
무선 충전 도킹 스테이션은 별도 구매 옵션입니다.
:::

## 개봉 시 주의사항

1. 포장 박스를 개봉할 때 로봇 본체에 흠집이 생기지 않도록 주의합니다
2. 로봇을 꺼낸 후 **비상정지 버튼이 눌려있는 상태**인지 확인합니다
3. 4개의 스워브 휠이 모두 정상적으로 지면에 닿는지 확인합니다
4. 외관에 운송 중 발생한 손상이 없는지 점검합니다
