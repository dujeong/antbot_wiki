**ANTBot**은 ROBOTIS AI에서 개발한 오픈소스 **실외 자율주행 배송 로봇**입니다.
4개의 바퀴가 각각 독립적으로 조향하는 **스워브 드라이브(Swerve Drive)** 방식으로,
전방향 이동과 제자리 회전이 가능하며, 라스트마일 배송에 최적화되어 있습니다.

## 구성품

기본 패키지에 포함된 구성품:

<Steps>

1. **ANTBot 본체** `x1`

   4WD 스워브 드라이브 로봇 플랫폼 (전체 센서 + 컴퓨팅 탑재)

2. **수동 충전기** `x1`

   HMI 패널 충전 단자에 연결하여 충전

3. **무선 충전 도킹 스테이션** `옵션`

   별도 구매, 자동 도킹 및 무선 충전

</Steps>

## 소프트웨어 스택

ANTBot의 전체 소프트웨어는 **9개의 모듈형 ROS 2 패키지**로 구성됩니다:

| 패키지 | 언어 | 역할 |
| :--- | :---: | :--- |
| `antbot` | - | 메타 패키지 |
| `antbot_bringup` | Python | 전체 시스템 launch 파일 |
| `antbot_description` | Xacro | URDF/Xacro 로봇 모델 |
| `antbot_swerve_controller` | C++17 | 스워브 드라이브 컨트롤러 (IK, 오도메트리) |
| `antbot_hw_interface` | C++17 | ANT-RCU ros2_control 플러그인 |
| `antbot_libs` | C++17 | Dynamixel Protocol 2.0 통신 |
| `antbot_camera` | C++17 | 멀티 카메라 드라이버 |
| `antbot_imu` | C++17 | IMU 드라이버 (상보 필터) |
| `antbot_teleop` | Python | 키보드/조이스틱 원격 조종 |
