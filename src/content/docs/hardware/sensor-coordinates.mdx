## 센서 구성

| 센서 | 모델 | 수량 | 연결 | 비고 |
| :--- | :--- | :---: | :--- | :--- |
| 3D LiDAR | Vanjee WLR-722 | 1 | Ethernet | 360° 포인트 클라우드 |
| 2D LiDAR | COIN D4 | 2 | USB Serial | 전방 + 후방 장애물 감지 |
| RGB-D 카메라 | Orbbec Gemini 336L | 1 | USB | Color + Depth |
| 모노 카메라 | Novitec V4L2 | 3 | V4L2 | 등거리 렌즈, 640×360 @ 30fps |
| 후방 카메라 | Generic V4L2 | 1 | V4L2 | 15 FPS |
| 적재함 카메라 | USB Camera | 1 | USB | 적재 상태 모니터링 |
| IMU | 6축 (RCU U60) | 1 | DXL Serial | |
| GNSS | u-blox | 1 | Serial | GPS 수신기 |
