---

## 플랫폼 기본 정보

| 항목 | 내용 |
| :--- | :--- |
| **ROS2 배포판** | Humble |
| **DDS 미들웨어** | FastDDS |
| **OS 및 커널** | Ubuntu 22.04 / 5.15.136-tegra |
| **JetPack** | 6.0 (L4T R36.3.0) |
| **부팅 시 자동 실행** | systemd 서비스명: `antbot` |
